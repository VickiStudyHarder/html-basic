Floating Element

another technique for positioning and creating layout

any independent, self contained piece of content

use clear: both on any element below to move it under the block

by the default, parent element don't see Floating element

1. add a div with clear to remove the effect, but the div is not descriptive

2. use pseudo code to apply clear, we can use a more general class name to apply the clearfix

3. set overflow to anything other then visible, i.e. hidden. rarely used for flex and grid

    grid property
  • grid-template-row: set the number of row
  • grid-template-column: set the number of column
  • grid-template: set the number of row/column

  • align-item: along the vertical axis, move the item in within the grid
  • justify-content: move the whole grid in the container horizontally
  • algin-content: move the whole grid in the container vertically

  • grid-column: set the starting column of grid item
  • grid-area: start row number / start column number / end row number / end column number

  • grid-template-areas: give a name to every space in the grid, . for empty
